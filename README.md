# Calculator – C# Windows Forms

تطبيق **آلة حاسبة بسيطة** تم تطويره باستخدام **C# و Windows Forms**.
يسمح بإجراء العمليات الحسابية الأساسية من خلال واجهة رسومية سهلة الاستخدام.

---

## 📌 المميزات

* إدخال الأرقام من **0 إلى 9**
* دعم **الأعداد العشرية**
* العمليات الحسابية الأساسية:

  * الجمع `+`
  * الطرح `-`
  * الضرب `x`
  * القسمة `/`
  * باقي القسمة `%`
* زر **=** لحساب النتيجة
* زر **CLS** لمسح الشاشة
* زر **DEL** لحذف آخر رقم
* التحقق من **القسمة على صفر** لمنع الأخطاء

---

## 🖥️ واجهة التطبيق

واجهة بسيطة تتكون من:

* **RichTextBox** لعرض العمليات والنتائج
* أزرار للأرقام
* أزرار للعمليات الحسابية
* أزرار للتحكم (CLS – DEL)

---

## ⚙️ طريقة عمل البرنامج

1. المستخدم يضغط على أزرار الأرقام.
2. يتم إضافة الرقم إلى مربع العرض.
3. عند الضغط على عملية حسابية يتم حفظ العملية.
4. بعد إدخال الرقم الثاني والضغط على **=**:

   * يتم تقسيم النص إلى رقمين.
   * يتم تنفيذ العملية المطلوبة.
   * يتم عرض النتيجة في مربع النص.

### 3️⃣ إدخال الأرقام

الدالة المسؤولة عن عرض الرقم عند الضغط على زر.

```csharp
void Numbers(string Num)
{
    if (isFirstInput)
    {
        richTextBox1.Clear();
        isFirstInput = false;
    }

    richTextBox1.Text += Num;
}
```

---

### 4️⃣ تنفيذ العمليات الحسابية

يتم تحليل النص لمعرفة العملية وتنفيذها.

مثال على الجمع:

```csharp
Parts = text.Split('+');
firstNum = Convert.ToDouble(Parts[0]);
secondNum = Convert.ToDouble(Parts[1]);

result = firstNum + secondNum;
```

## 🚀 كيفية تشغيل المشروع

1. افتح المشروع باستخدام **Visual Studio**
2. تأكد أن نوع المشروع **Windows Forms**
3. اضغط **Start (F5)**
4. ستظهر واجهة الآلة الحاسبة ويمكنك استخدامها مباشرة.
