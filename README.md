
# Simple Calculator Application

## Overview

Calculator Application is a simple desktop calculator built using **C#** and **Windows Forms**.
It performs basic arithmetic operations and provides a clean graphical interface for user interaction.

The calculator allows users to enter numbers, choose an operation, and view the result instantly.

---

# Features

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (/)
  * Modulus (%)
* Support for decimal numbers
* Delete last entered character
* Clear the entire input
* Error handling (example: division by zero)
* Dynamic UI color mode (Black / Red theme)

# How It Works

### Number Input

Numbers are inserted into the display using buttons.
Each button sends its value to the `Numbers()` function which updates the display.

### Operations

The calculator supports:

* `+`
* `-`
* `x`
* `/`
* `%`

When an operation button is pressed, it is stored and used during calculation.

### Calculation

When the **Equal (=)** button is clicked:

1. The input text is split based on the operator.
2. The first and second numbers are extracted.
3. The corresponding arithmetic operation is executed.
4. The result is printed in the display.

Example:

```
5 + 3
= 8
```

# Error Handling

The application handles common errors such as:

* Division by zero
* Invalid input
* Starting with an operator

Example:

```
10 / 0
Error
```
